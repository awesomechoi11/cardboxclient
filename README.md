## Key Features

-   **Card Pack Creator Page**
    -   Debounced autosave
    -   Rich Text Editor
    -   Drag & Drop rearrange
-   **Card Pack Viewer**
    -   Card Swiper
        -   Drag & Drop
            -   Optimized animations
-   **Card Pack Gamemode**
    -   Match
        -   Accurate Timer
            -   Interactive Animations
-   **Modals**
    -   Image Picker - Image Uploader connected to CDN and DB - Drag & Drop uploader - Image Url Uploader
        -   Login/ Signup
            -   MongoDB login
                -   Email Confirmation
        -   Publish Confirmation
-   **Styles**
    -   SCSS
        -   Tailwind
        -   Bootstrap
-   **Analytics**
    -   Umami
        -   Google
